Decisão Estratégica: PT Only

O agente opera exclusivamente em Principal Tokens (PT) — yield fixo garantido até maturidade. YT (Yield Token) foi excluído do escopo para capital de usuários por razões de produto:

  • YT é uma posição alavancada e especulativa — pode ir a zero no vencimento
  • Incompatível com a proposta DollarUp: "rendimento passivo, sem risco de perda do principal"
  • PT garante yield fixo independente de flutuações do underlying APY
YT pode ser considerado futuramente apenas para treasury própria da DollarUp (fees acumulados), nunca para capital de usuários.

Fluxo de Execução por Ciclo (15 min)

# 1. Risk scan externo (Perplexity — 1x/dia às 06h)
risk = perplexity_risk_scan(['Pendle', 'sUSDai', 'Morpho', 'Aave'])
if risk.severity == 'CRITICAL': trigger_circuit_breaker()

# 2. Observe — dados on-chain e API
snapshot = fetch_pendle_market_data(market_id)  # API v1 Arbitrum
position = get_current_pt_position(safe_address)
portfolio = get_portfolio_total(safe_address)

# 3. Pre-validate — filtros obrigatórios
validation = pre_validate(snapshot, asset_ctx)
if not validation.passed: continue

# 4. Quant Math Engine — ΔNet e threshold
cost      = compute_ctea(gas, slip_in, slip_out, notional, dtm)
delta_net = compute_delta_net(snapshot, cost)
threshold = compute_dynamic_threshold(profile, yvi)

# 5. Claude — classifica regime e ajusta regime_bias
regime = claude_classify_regime(snapshot_dict)
bias   = regime.regime_bias  # 1.0 se fallback ativo

# 6. Decisão
if   delta_net < -(threshold * bias): action = ROTATE_TO_PT
elif should_exit(snapshot, position):  action = EXIT_PT
else:                                   action = HOLD_NO_OP

# 7. Execute via Safe (telemetria gravada ANTES da TX em LIVE)
emit_telemetry(snapshot, decision, state)  # atomicidade garantida
result = execute(decision, state)          # DRY_RUN ou LIVE

Lógica Quantitativa

5.1 CTEA Anualizado

CTEA = (GasUSD + SlippageEntry + SlippageExit) / NotionalUSD × (365 / DTM)

Todos os custos pontuais (gas e slippage) são anualizados pelo prazo de vencimento para comparação dimensional correta com APY.

5.2 ΔNet

ΔNet = ΔTime − CTEA − RiscoSaídaAnualizado
ΔTime = fixed_apy − implied_apy
RiscoSaída = 0.005 (0.5% a.a. conservador)

5.3 YVI — Yield Volatility Index

σ24h = desvio padrão amostral dos ΔNet horários das últimas 24h
Δtarget_effective = Δtarget_base + (k × σ24h)

5.4 Árvore de Decisão PT

# Prioridade 1: Saída protetiva (verifica antes de qualquer entrada)
if fixed_apy < min_fixed_apy:                   → EXIT_PT  # APY colapsou
if tvl_usd < min_tvl × 0.3:                    → EXIT_PT  # TVL deteriorou
if days_to_maturity < 7 and position > 0:      → EXIT_PT  # Vencimento próximo

# Prioridade 2: Entrada — mercado superprecifica yield
if delta_net < -(threshold × regime_bias):      → ROTATE_TO_PT

# Prioridade 3: Inércia — banda efetiva
if -threshold ≤ delta_net ≤ threshold:          → HOLD_NO_OP  # Caso C

Camada AI (ai_advisor.py)

Claude — Classificador de Regime

  • Modelo: claude-sonnet-4-20250514
  • Input: snapshot JSON (fixed_apy, implied_apy, ΔNet, TVL, DTM, YVI)
  • Output: regime, confidence, regime_bias, reasoning
  • Fallback determinístico se confidence < 0.65 → regime_bias = 1.0
  • Remove markdown automático (Claude retorna com backticks)

Cenários de Teste (PT Only)

Caso Condições Comportamento Esperado Flag
A: PT Atrativo fixed_apy >> implied_apy, CTEA baixo, TVL saudável Compra PT até limite do pool ROTATE_TO_PT
B: Zona Inércia ΔNet ≈ 0, YVI alto alargando bandas Mantém posição atual inalterada HOLD_NO_OP
C: Exit Protetivo fixed_apy abaixo do mínimo ou TVL colapso Sai do PT para proteger capital EXIT_PT
D: Risco Externo Perplexity severity=CRITICAL Bloqueia novas TXs, mantém posição CIRCUIT_BREAKER
E: Falhas Acumuladas 5+ erros em 60 minutos Trava agente — reset manual CRITICAL_HALT
F: Fallback AI confidence < 0.65 ou Claude indisponível Modo determinístico, regime_bias=1.0 FALLBACK_DETERMINISTIC
